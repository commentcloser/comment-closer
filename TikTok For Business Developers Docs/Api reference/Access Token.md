Get, renew or revoke a Creator access token
To get or renew the short-term Creator access token, you can use /tt_user/oauth2/token/ and /tt_user/oauth2/refresh_token/ respectively.
The /tt_user/oauth2/token/ endpoint now requires redirect_uri for better security. The short-term Creator access token remains valid for 24 hours, whereas the Creator refresh token remains valid for a year. You can get the access token using /tt_user/oauth2/token/ and refresh it daily with the refresh token via /tt_user/oauth2/refresh_token/ within one year. However, after one year, you need the creator to reauthorize it.
To revoke a Creator access token, use /tt_user/oauth2/revoke/.

Authentication
After you get auth_code from advertisers, TikTok account users, TikTok Creator Marketplace (TTCM) accounts, or TikTok One (TTO) Creator Marketplace accounts, you can use it to generate an access token for subsequent API requests.

The specific steps required for authentication will vary depending on the type of API you are using. Please refer to the following topics for detailed instructions on how to complete authentication：

Marketing API authentication: Refer to Marketing API authentication to learn how to generate a long-term access token. This access token does not expire.
Accounts API / Mentions API authentication: Refer to Accounts API authentication or Mentions API authentication to learn how to generate an access token. The access token is valid for one day. Once an access token has expired, you need to renew the access token via /tt_user/oauth2/refresh_token/.
TTO API authentication:
Refer to Obtain authentication from the TTO Creator Marketplace account to learn how to generate an access token for TTO Creator Marketplace accounts. This access token does not expire.
Creator Marketplace API authentication:
Refer to TTCM account authentication to learn how to generate an access token for TTCM accounts. This access token does not expire.
Refer to Get authorization from Creators - Step 3 to learn how to generate a Creator access token. It is valid for one day. Once an access token has expired, you need to renew the access token via /tt_user/oauth2/refresh_token/.
Discovery API authentication: Refer to Discovery API authentication to learn how to generate an advertiser access token or TikTok account access token. This advertiser access token does not expire while the TikTok account access token is valid for one day. Once the TikTok account access token has expired, you need to renew the access token via /tt_user/oauth2/refresh_token/..
Spark Ads Recommendation API authentication: Generate a TTCM account access token or TikTok account access token. This TTCM account access token does not expire while the TikTok account access token is valid for one day. Once the TikTok account access token has expired, you need to renew the access token via /tt_user/oauth2/refresh_token/.
